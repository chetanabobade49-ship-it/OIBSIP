🔐 MediCare Patient Login Authentication System

This is a simple **Login & Registration Authentication Web App** for a patient healthcare portal called **MediCare**. It is built using **HTML, CSS, and JavaScript** and includes a login system, registration system, and a patient dashboard UI.

---

🚀 Features

- Patient Registration page
- Patient Login page
- Form validation (password confirmation, required fields)
- Dashboard after login
- Patient profile display (name, age, email, etc.)
- Logout functionality
- Simple UI for healthcare portal

---

🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript 

---

🔑 How It Works

1. User registers with details (name, username, password, etc.)
2. Data is stored (usually in localStorage in simple projects)
3. User logs in using username and password
4. If credentials match → redirected to dashboard
5. User can logout anytime

---

💡 Learning Outcomes

- Form handling in JavaScript  
- Basic authentication logic  
- DOM manipulation  
- Page navigation  
- UI design for dashboard systems
