🌟 Tribute Page – Dr. A. P. J. Abdul Kalam

This is a simple tribute webpage dedicated to **Dr. A. P. J. Abdul Kalam**, the “Missile Man of India” and former President of India. It is built using **HTML and CSS** and highlights his life, achievements, and inspirational journey.

---
🚀 Features
- Clean tribute page layout  
- Biography of Dr. A. P. J. Abdul Kalam  
- “What I Admire About Him” section  
- Inspirational quote  
- Books written by and about him  
- Wikipedia reference link  
---
🛠️ Technologies Used
- HTML5
- CSS3
